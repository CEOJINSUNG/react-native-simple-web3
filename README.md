# react-native-simple-web3

Web3.js for react-native

## Installation

```sh
npm install react-native-simple-web3
```

## Usage

```js
import SimpleWeb3 from "react-native-simple-web3";

// ...

const result = await SimpleWeb3.multiply(3, 7);
```
